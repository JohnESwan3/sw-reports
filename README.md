# SW-Reports
## Purpose
This application is used to import csv files that are exported from SolarWinds. The reports have custom data.
* Monitor ticket load for Service Desk agents
* Report on new hire onboarding metrics (Day 1 and Day 3)

## Stack
* Rust
* Iced UI Library
* SQLite (sqlx)
* Plotters + Plotters Iced (charts and graphs)
